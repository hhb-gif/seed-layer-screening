# 种子层筛选 - 配置 UI 设计文档

## 概述

为种子层筛选项目构建 Streamlit 配置 UI，支持：编辑配置参数 → 启动 pipeline → 查看评分结果和图表。

**范围**：本地单用户使用，单配置文件，后台执行 pipeline。

## 功能模块

### 1. 配置编辑器（侧边栏）

每个配置区块用 `st.expander` 折叠，内部用对应控件：

| 区块 | 控件类型 | 说明 |
|------|---------|------|
| 工作离子 | `st.selectbox` | Li/Zn/Mg/Na/K/Ca，改了自动更新 ref_miller |
| 材料池初筛 | `st.slider` + `st.multiselect` | 阈值滑块 + 排除元素多选 |
| 晶格匹配 | `st.slider` + `st.number_input` | 失配 %、缩放倍数 |
| 表面模型 | `st.number_input` | 真空层、slab 厚度 |
| 势函数 | `st.selectbox` + `st.checkbox` | MACE/CHGNet、CPU/CUDA、D3 校正 |
| 弛豫参数 | `st.number_input` | fmax、steps（bulk/slab/adsorb 各一组） |
| 界面能 | `st.number_input` + `st.slider` | 最大层数、fmax、steps |
| 吸附能 | `st.slider` + `st.number_input` | 覆盖度、能量窗口 |
| NEB 扩散 | `st.number_input` + `st.checkbox` | images、fmax、climb |
| 评分权重 | `st.slider`（5 个） | 总和自动归一化显示 |
| 输出控制 | `st.checkbox` | 保存结构/轨迹 |

**关键交互**：
- 工作离子改了 → ref_miller 下拉选项联动（Li→[1,1,0]，Zn→[0,0,1] 等）
- 评分权重改了 → 实时显示归一化后的百分比
- 「保存配置」→ 覆写 `configs/default.yaml`，显示成功提示

**后台执行机制**：
- 用 `subprocess.Popen` 启动 `python src/main.py --config ... --tag ...`
- `st.session_state` 存储 `Popen` 对象，刷新页面不丢失进程
- 每 2 秒轮询 `output/<run_id>/logs/run.log` 最后 20 行显示
- 进程结束时自动检测（`process.poll()`），切换到「已完成」状态
- 停止按钮：`process.terminate()`

## 技术实现

### 文件结构

```
seed-layer-screening/
├── app.py                  ← Streamlit 入口（新增）
├── src/
│   └── seed_layer/
│       └── config.py       ← 加 save_config() 函数（小改）
└── configs/
    └── default.yaml        ← 被 UI 读写
```

### 依赖

```toml
# pyproject.toml 添加
streamlit = ">=1.30"
```

### 启动方式

```bash
streamlit run app.py --server.port 8501
```

### 关键实现

| 功能 | 实现方式 |
|------|---------|
| 配置读取 | 复用 `load_config()` |
| 配置保存 | 新增 `save_config(config_dict, path)` 写 YAML |
| 后台执行 | `subprocess.Popen(["python", "src/main.py", ...])` |
| 日志轮询 | `st.empty()` + `time.sleep(2)` 循环读 `run.log` |
| 进程管理 | `st.session_state` 存 `Popen` 对象 |
| 结果读取 | `pd.read_csv(summary.csv)` + `st.dataframe` |
| 图表展示 | `st.pyplot(fig)` 或读 PNG `st.image()` |
| 历史运行 | 扫描 `output/` 目录下的子目录列表 |

### 配置保存函数（config.py 新增）

```python
def save_config(config_dict: dict, path: str) -> None:
    """Save config dict to YAML file."""
    import yaml
    with open(path, "w", encoding="utf-8") as f:
        yaml.dump(config_dict, f, default_flow_style=False, allow_unicode=True)
```

## 不做的事

- 不做多配置预设管理
- 不做用户登录/权限
- 不做远程访问部署
- 不做 pipeline 断点续跑 UI（CLI 已有 `--resume` 预留）
- 不做实时进度条（只显示日志）
